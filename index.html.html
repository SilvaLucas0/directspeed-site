Política de Privacidade

Esta política explica quais dados o DirectSpeed processa, por que eles são usados e como o usuário pode gerenciar essas informações.

Última atualização: 06/07/2026

1. Finalidade da extensão

O DirectSpeed tem como finalidade auxiliar o usuário a organizar, executar e acompanhar rotinas de prospecção comercial no Instagram por meio de painel flutuante, controle de leads, templates de mensagens, ciclos, pausas, metas, limites e histórico operacional.

2. Informações coletadas ou processadas

A extensão pode processar as seguintes informações:

  • E-mail informado pelo usuário para validação de acesso;
  • Configurações da campanha, como mensagens, delays, metas, ciclos e limites;
  • Lista de leads inserida manualmente ou importada de planilha compartilhada pelo usuário;
  • Histórico local de ações, incluindo status de leads, envios, erros e registros operacionais;
  • Link de planilha do Google Sheets informado pelo usuário, quando usado para importação de leads.

3. Como as informações são usadas

As informações são usadas exclusivamente para validar o acesso do usuário, salvar preferências locais, carregar e organizar leads informados pelo usuário, executar e acompanhar a rotina configurada, além de exibir métricas, status, histórico e logs operacionais dentro da própria extensão.

4. Compartilhamento de dados

O e-mail informado pelo usuário pode ser enviado ao endpoint de validação configurado pelo desenvolvedor para confirmar se o acesso está liberado. A extensão também pode acessar planilhas compartilhadas pelo próprio usuário quando ele informa um link do Google Sheets para importação de leads.

O DirectSpeed não vende dados do usuário, não transfere dados para plataformas de anúncios e não utiliza os dados para publicidade personalizada.

5. Armazenamento local

Configurações, templates, leads e histórico operacional podem ser armazenados localmente no navegador por meio da API chrome.storage.local, com a finalidade de manter o funcionamento da extensão entre sessões.

6. Permissões da extensão

A extensão solicita permissões para armazenamento local, acesso à aba do Instagram, inserção do painel, agendamento de pausas e importação de planilhas quando informado pelo usuário. Essas permissões são usadas apenas para entregar a funcionalidade principal da extensão.

7. Segurança

A extensão foi projetada para limitar o uso de dados à sua finalidade principal. O usuário deve utilizar a ferramenta em conformidade com as regras das plataformas acessadas e com a legislação aplicável.

8. Retenção e exclusão

Dados salvos localmente podem ser removidos pelo usuário ao limpar os dados da extensão no navegador ou ao desinstalar a extensão.

9. Contato

Para dúvidas sobre esta Política de Privacidade, entre em contato pelo e-mail contato@directspeed.app.

10. Declaração de uso limitado

O uso das informações recebidas pela extensão estará de acordo com a Política de Dados do Usuário da Chrome Web Store, incluindo os requisitos de uso limitado.

Resumo: os dados são usados apenas para validar acesso, salvar preferências, organizar leads e permitir o funcionamento principal do DirectSpeed.
